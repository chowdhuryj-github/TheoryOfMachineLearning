Another metric used to quantify error is the mean squared error. The MSE value provides a way to analyze the accuracy of the model. It measures the average average squared difference between the predicted and actual values in the dataset.

Some Advantages & Disadvantages of MSE:

  • Advantage: the MSE leads to a stable convergence of the model, as the mean square difference between predictions and actual values is minimized
  • Disadvantage: if datasets contain outliers that the model can't predict well, the MSE may be a poor loss function as it doesn't accurately reflect the predictive quality of the model

Some Advantages & Disadvantages of MAE:

  • Advantage: MAE is very robust regarding outliers, as MAE tends to calculate the absolute differences between the predicted and actual values
  • Disadvantage: MAE focuses more on the magnitude of the error rather than its direction, which means it can neglect systemic bias of the model

In MAE, the error function is piecewise linear with respect to each parameter. For the MSE, the error function is quadratic with respect to each error. Keeping the advantages and disadvantages mentioned above in mind, the shapes make sense because

  • For MAE, there can be flat regions around small errors, which means slower convergence
  • For MSE, there can be smooth parabolas, which means it is very sensitive to outliers

The Gaussian Distribution Probability Density Function

$$f(x) = \frac{1}{\sigma\sqrt{2\pi}} e^{-\frac{(x - \mu)^2}{2 \sigma^2}}$$

In this probability density function, the model parameters are $\mu$ and $\sigma$ The independent variables are x and the dependent variables are f(x).

def gaussian_pdf_plot(input_val: np.ndarray, mu: float, sigma: float) -> np.ndarray:

    """
        this is a method that returns a numpy array of the predictions being made
    """

    part_one = 1 / (sigma * np.sqrt(2 * np.pi))
    part_two = -1 * (input_val - mu)**2 / (2 * sigma**2)
    final_val = part_one * (np.exp(part_two))
    
    return final_val

The Mean Absolute Error Formula

$$ MAE = \frac{1}{n} \sum_{i=1}^{n} | x_i - x | $$

# defining a function that calculates the mean absolte error

def compute_mae(array_one: np.ndarray, array_two: np.ndarray) -> float:

    """
        this is a method that computes the mean absolute error and returns it as a float
    """

    difference = np.abs(np.subtract(array_one, array_two))
    difference_sum = np.sum(difference)
    mean_square_error = (1 / len(array_one)) * difference_sum
    return mean_square_error
